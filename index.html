Projetos de Ciência de Dados

Predição de Vendas para Rede de Farmácias Rossmann

O CFO de uma grande rede de farmácias pretende realizar um grande investimento em seus estabelecimentos farmaceuticos, para isso, necessitava da previsão de vendas para os próximos 6 meses de cada loja, para então decidir o quanto investiria em cada loja, baseado na projeção de vendas. Então, realizei a criação de um modelo utiliando machine leaning para realizar a previsão das vendas, além disso, para facilitar o acesso a essa informação, crirei um bot no telegram que faz a captura das predições e a informa ao usuário.

Ferramentas utilizadas:
  • Python, Pandas, Numpy, Seaborn.
  • Boruta, Sklearn, Scipy.
  • Linear Regression, Lasso, Random Forest Regressor, XGBoost Regressor.
  • Regression Metrics: RMSE, MAE, MAPE.
  • Telegram Bot, Flask, Heroku Cloud.
